# AI Security Reviewer

AI Security Reviewer is a web service that automates security reviews for web applications developed with Spec-Driven Development (SDD).

## Repository Structure

- `frontend/`: Frontend application workspace (React + TypeScript + Vite)
- `backend/`: Backend API workspace (Python + FastAPI)
- `infra/`: Infrastructure as code workspace (Bicep)
- `docs/`: Mockups and supporting documentation
- `specs/`: Specification, implementation planning, and task artifacts
