検出されたコード

src/api/users.js ⚠ 脆弱性検出 
40  app.get('/users/:id', (req, res) => {
41    const id = req.params.id;
42    const query = `SELECT * FROM users
43      WHERE id = ${id}`;  // ⚠ 危険
44    db.query(query, (err, rows) => {
45      res.json(rows);
46    });
47  });

AI による解説

ユーザーから受け取った req.params.id を直接SQL文字列に連結しているため、攻撃者が任意のSQLを実行できる状態です。

攻撃例
GET /users/1 OR 1=1
→ 全ユーザー情報が漏洩する可能性

プリペアドステートメント（パラメータ化クエリ）を使用することで防止できます。

修正案

src/api/users.js 
40  app.get('/users/:id', (req, res) => {
41    const id = req.params.id;
42    const query = 'SELECT * FROM users WHERE id = ?';
43    db.query(query, [id], (err, rows) => {
44      res.json(rows);
45    });
46  });
