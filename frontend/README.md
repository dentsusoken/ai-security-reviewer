# frontend

Frontend workspace initialized with Vite + React + TypeScript.

## Stack

- React
- TypeScript (strict mode)
- Vite
- Tailwind CSS (with PostCSS + Autoprefixer)
- react-router-dom
- @azure/msal-browser
- @azure/msal-react

## Setup

```powershell
npm install
```

## Run (Dev)

```powershell
npm run dev
```

Default dev URL is shown in terminal (typically `http://localhost:5173`).
